# Fitness-app

**Fitness-app** - веб-приложение, которое позволяет подобрать для себя подходящий курс тренировок (йога, фитнес, бодифлекс и т.д), выполнять упражнения вместе с тренером, а также фиксировать свой прогресс прохождения.

## Основные возможности:
- Авторизация и регистрация — безопасный вход и создание личного аккаунта.
- Обзор всех курсов — на главной странице доступен список доступных тренировочных программ.
- Детальный просмотр курса — каждому курсу есть подробная информация, описание.
- Добавление курсов — возможность добавлять интересующие курсы.
- Профиль пользователя — тут отображаются личные данные и список добавленных курсов с прогрессом прохождения.
- Прогресс тренировок — для каждого курса отображается текущий прогресс выполнения тренировок.
- Список тренировок — при выборе курса открывается список всех занятий.
- Страница тренировки — просмотр видео с упражнениями, отметка выполненных упражнений и отслеживание прогресса.
- Обновление прогресса — запись и обновление выполненных упражнений прямо на странице тренировки.

## Технологии

В проекте использованы следующие основные технологии и библиотеки:

- **React (v19.1.0)** — библиотека для создания пользовательских интерфейсов
- **Next.js (v15.5.6)** — фреймворк для серверного рендеринга и статической генерации сайтов на React
- **React Redux (v9.2.0)** — управление состоянием приложения
- **Redux Toolkit (v2.9.1)** — официальный набор инструментов для Redux
- **TypeScript** — статическая типизация JavaScript
- **axios (v1.12.2)** — HTTP клиент для взаимодействия с API
- **Tailwind CSS (v4.1.14)** — утилитарный CSS-фреймворк для быстрого стилизации
- **PostCSS (v8.5.6)** и **@tailwindcss/postcss (v4.1.14)** — инструменты для обработки CSS
- **react-player (v3.3.3)** — компонент для вставки видео/аудио
- **react-toastify (v11.0.5)** — уведомления в приложении

## Установка и запуск проекта

### Требования

- Node.js (рекомендуется версия 16+)
- npm 

### Установка зависимостей

### Клонируйте репозиторий
git clone https://github.com/Pavekatal/fitness-app.git

### Перейдите в папку проекта
cd имя-проекта

### Установите зависимости
npm install  

### Для сборки проекта используйте команду:
npm run build
